

# Backend for the ChatBot using Langchain
## Setup
1. Clone the project 

2. ```cmd 
   cd Air-Ai-Flask
    ```
    
3. ```cmd 
   pip install virtualenv 
    ```  
    
4. ```cmd  
    virtualenv -p python3 venv 
    ```  
    
5. ```cmd   
    source venv/bin/activate 
   ```  

6. ```cmd  
    pip install -r requirements.txt
    ```
